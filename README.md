# In4graphic-labeling tool

Project structure:

```bash
project/
├── backend/
│   └── main.py
├── frontend/
│   └── app.py
└── data/
    └── images/
    └── data.json
```

Data

```bash
mkdir data
```

extract the `data.json` and `images` folder to the `data` folder.

## Backend

```bash
cd backend
python main.py
```

## Frontend

```bash
cd frontend
streamlit run app.py
```
